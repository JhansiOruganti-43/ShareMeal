# 🤝 Contributing to ShareMeal

Thank you for your interest in contributing to ShareMeal!

We welcome contributions from everyone. Please follow the guidelines below before submitting your work.

---

## 🚀 Getting Started

1. Fork this repository.
2. Clone your fork to your local machine.

```bash
git clone https://github.com/<your-username>/ShareMeal.git
```

3. Install the project dependencies.

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🌿 Create a Branch

Create a new branch before making changes.

```bash
git checkout -b feature/your-feature-name
```

Example:

```bash
git checkout -b fix/notification-badge
```

---

## 💻 Coding Guidelines

- Write clean and readable code.
- Follow the existing project structure.
- Keep pull requests focused on one issue.
- Test your changes before submitting.

---

## 📝 Commit Messages

Use meaningful commit messages.

Examples:

```
fix: clear notification badge after viewing
feature: add QR code verification
docs: improve README
```

---

## 🔀 Pull Requests

Before submitting a Pull Request:

- Ensure the application builds successfully.
- Link the related GitHub Issue.
- Provide screenshots for UI changes.
- Describe your implementation clearly.

---

## 🐞 Reporting Bugs

When reporting a bug, please include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots (if applicable)

---

## 💡 Suggesting Features

Feature requests should include:

- Problem statement
- Proposed solution
- Expected outcome

---

## ❤️ Thank You

Every contribution helps make ShareMeal better.

Happy Coding! 🚀
